Проект по DirectX 11 (C++)

Изучил создание графического Windows приложения, создание окна приложения, настройку DirectX 11 для отрисовки графики.

Реализовал свободную камеру с возможностью управлять ею клавиатурой и мышью.

Реализовал визуализацию 3D-моделей в 3D пространстве с возможным применением трансформаций (смещение, поворот, масштабирование).

Реализовал абстракцию над Window Messages для хранения информации о действиях пользователя (клавиатура и мышь), реализовал возможность добавить функцию-обработчик на любые его действия.
